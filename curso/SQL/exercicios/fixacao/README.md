## Sobre os exercícios

Os exercícios de fixação são criados com a ajuda de inteligência artificial, com base nas minhas anotações e momento de estudo.

Costumo pedir exercícios mais simples e, depois, mais elaborados — onde eu possa usar o exercício para fixar algo novo e também colocar em prática conceitos "antigos".

Além disso, os exercícios contêm comentários com as correções feitas pela IA.

O **16-04_Exercicio5** é o exemplo perfeito de como eu "resolvi" o problema, mas poderia ter feito de uma maneira muito melhor e mais fácil, mas não saberia sem a corração.
