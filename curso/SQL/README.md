# SQL com SQLite

Repositório criado durante meus estudos de SQL utilizando o curso do Teo Me Why.

Curso utilizado:

* https://www.youtube.com/watch?v=VmkJG8awKqM&list=PLvlkVRRKOYFRo651oD0JptVqfQGDvMi3j

## Objetivo

Consolidar os fundamentos de SQL por meio de anotações, exercícios práticos e resolução de problemas utilizando SQLite.

Este repositório documenta minha evolução no aprendizado de:

* Modelagem e estrutura de bancos relacionais
* Consultas SQL
* Manipulação e filtragem de dados
* Agregações e análises exploratórias
* Boas práticas de consulta

## Estrutura do Repositório

```text
sql/
```
